SkyMonitor provides accurate weather information using NWS, SPC, PirateWeather, and OpenWeather APIs, and it provides a modern-styled interface featuring weather and civil alerts, forecasts, radar, SPC outlooks, a minute-by-minute precipitation forecast, and much more. My goal with SkyMonitor is to make a website in the style of a modern weather app, but with more accuracy and data like an official NWS website.
